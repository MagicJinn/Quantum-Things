# Link Orb

## Description

---

The Link Orb is a decorative block that can be used to create links between different locations or systems. It is owned by the player who places it and appears as a small floating orb.

The Link Orb serves as a connection point for various linking mechanisms and can be used in conjunction with other blocks to create teleportation or connection networks. Its exact functionality depends on the systems it is connected to.
