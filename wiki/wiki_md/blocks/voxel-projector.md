# Voxel Projector

## Description

---

The Voxel Projector is a block that allows you to project and display custom voxel-based structures. Right-clicking the block opens a GUI where you can configure the projection settings and design your voxel structure.

The projector can be used for decorative purposes, creating custom displays, or building complex visual effects. The exact functionality and features are configurable through the block's interface.
